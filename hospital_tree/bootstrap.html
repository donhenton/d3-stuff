Homepage

This application is a demonstration system for various web technologies. The main technologies on exhibit are :
  1. Spring MVC 3.0
  2. JQuery for JSON processing
  3. JQuery Functionality
  4. Twitter Bootstrap UI Elements
  5. Backbone.js for Javascript MV*
Apache Tiles is used to format the pages, allowing for reusable sections of markup.
